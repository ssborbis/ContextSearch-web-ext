Help
Adding Search Engines
Right-click the search bar on any webpage and select Add Custom Search. New engines can also be added by hand in the Search Engines manager


Context Menu
Opening
Highlight a word and right-click to open the context menu. Expand the ContextSearch menu labeled Search for "..." and choose which search engine to perform the search with.


Quick Menu
Opening
Highlight a word and hold down the right mouse button until the Quick Menu appears. You can choose from several options for how the quick menu opens


Resizing and Reordering
The Quick Menu can be resized by changing the Columns and Rows setting or by dragging the resize icon in the lower-right corner of the menu. You can rearrange search engines using drag & drop


Toolbar
Opening
Access engines any time using the Toolbar button


Search Engines Manager
Adding Folders
Search engines can be organized in folders and subfolders.

1. Use the right-click menu and choose New Folder
2. Give the folder a unique name
3. Drag & drop search engines into the folder


Editing Icons
You can use custom icons for search engines

1. Double-click a search engine or right-click and choose Edit
2. Edit the Icon URI field with the url of the new icon and click Save


Importing Firefox Search Engines
As of Firefox 63, One-Click Search Engines are imported into ContextSearch automatically. However they cannot be modified unless they are imported through the file browser.

If you would like modify One-Click Search Engines, you can import them by clicking the button under 'Import One-Click Search Engines' in the Search Engines tab. Browse to your profile folder and select the search.json.mozlz4 file. You can find your profile path by opening about:profiles in the address bar. Typical locations:

  Windows

%APPDATA%/Mozilla/Firefox/Profiles/<profile>/search.json.mozlz4

  Mac OS X

Users/<username>/Library/Application Support/Firefox/Profiles/<profile>/search.json.mozlz4

  Linux

/home/<username>/.mozilla/firefox/<profile>/search.json.mozlz4
